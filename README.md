# SWOT-fine-scale-material-transport

This repository contains the MATLAB (R2023b) codes (*.m files) and the figure source data (.mat files) for the paper "Hidden Fine-Scale Transport Pathways and Biological Connectivity Revealed by SWOT".

For installation of MatLab, please refer to: https://au.mathworks.com/products/matlab.html

The netcdf4 file "GEBCO_2024_sub_ice_topo.nc" is too large to upload. Users can download it at this website:https://download.gebco.net/
